# Survive-The-JumpScare
A Horror Survial game made in unity using C# language. This games explores the story of a man whose car breaks down in front of an abandoned house. With no other way out he decides to explore the house to escape and learns the truth the hard way.


The full game cannot be uploaded in this repositry so you can download the game from the following link and play
